## English

A lightweight, secure, and fully responsive **web-based calculator** built using **jQuery** and semantic HTML5/CSS3. It features an event-driven computation architecture wrapped in a modern light frosted-glass (Glassmorphism) layout.

### 🔗 Live Preview
You can test the live version of this project here:  
👉 **[Launch Live Demo](https://demo.aminarjmand.com/Online-Jquery-Calculator/)**

### ✨ Key Features
- **Event-Driven UI:** Replaced resource-heavy processing loops with lightweight input event triggers.
- **Smart Arithmetic Validation:** Implemented strict error handling to prevent mathematical edge-case bugs like "Division by zero".
- **Decimal Optimization:** Automatically formats long fractional results up to 4 decimals for cleaner readouts.
- **Modern Layout:** Polished frosted-glass interface utilizing native backdrop CSS filtering and the Inter font family.
- **Client-Side Security:** Executes mathematical evaluations safely without using unsafe global evaluation handlers.
